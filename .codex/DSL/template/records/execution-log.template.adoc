== Log Meta

- log.id（日志ID）: LOG-XXXX
- log.status（日志状态）: DRAFT
- log.created_at（日志创建时间）: YYYY-MM-DD HH:MM:SS
- log.owner（日志负责人）: <OWNER>
- flow.id（流程ID）: TASK_EXECUTION_FLOW
- flow.version（流程版本）: 1.0

== Task Context

- task.id（任务ID）: TASK-XXXX
- task.name（任务名称）: <TASK_NAME>
- task.status（任务状态）: READY
- task.file（任务文件）: <TASK_FILE_PATH>
- task.scope.modules（范围模块）:
  - <MODULE_1>
- task.scope.excludes（范围排除）:
  - <EXCLUDE_1>
